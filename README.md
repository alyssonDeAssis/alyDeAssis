# alyDeAssis

App utilizando Api de Consulta do Github;

+Instalação dos pacotes:
cd backend
npm install

cd frontend 
npm install

-inicialização da aplicação:
cd backend 
npm start 

cd frontend 
npm start

+Tecnologias utilizadas:
React;
React-router-dom;
simulação de Api Rest utilizando o json-server;
axios para as HTTP requests;
Fonte: font-awesome;
CSS Grid;
Flexbox;
Bootstrap.
